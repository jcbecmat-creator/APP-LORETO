¿Cómo se calculan estos rubros?

Cálculo para Turno Diurno

El Salario Base Diurno se obtiene dividiendo tu Salario Semanal entre 6 días laborables. El Costo por Hora se calcula dividiendo ese salario base diario entre una jornada normal de 8 horas.

Cálculo para Turno Nocturno

El Salario Base Nocturno se obtiene dividiendo tu Salario Semanal entre 5 días laborables (debido a la compresión de la jornada). El Costo por Hora se calcula dividiendo ese salario base diario entre las 8 horas reglamentarias.
